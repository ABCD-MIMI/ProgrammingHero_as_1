HTML

HTML (HyperText Markup Language) is the foundational language. HyperText Markup Language is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content.

CSS

CSS (Cascading Style Sheets) is used to style web pages by applying styles to HTML elements. It allows you to control the layout, colors, fonts, and overall appearance of a website, ensuring a consistent and visually appealing design.

JAVASCRIPT

JavaScript (JS) is a versatile programming language used to create interactive effects within web browsers. It enables dynamic content such as form validation, animations, and interactive maps, making web pages more engaging and user-friendly.
